Auto Generate Lot/Serial number in Workorder in Manufacturing


Auto Generate Lot/Serial Number Configuration based on the Today date/ Production Date. And Allow Workorder form Settings.

Select product and allow Tracking by Lot or Serial number.

Create BOM and add product in BOM.

Create Manufacturing Order and select BOM of Relavent..

Create Workorder of Manufacturing Product.

Click on Smart button of work Orders.

Click on Start Working of Work orders and here Lot/Serail Numbers is Created. Here Lot/Serial number will be Create based on the sequence you have assigned in the Configuration based on the Today date and Date of Production.

Check Here Lot/Serial Number is Created for Workorder.


Auto Generate Lot/Serial number in Incoming Shipment
